Detailed Description

This is a specialization of the parser class for the value type erasure. The parser tries to identify the value provided by the data string and converts it to the appropriate type which will then be wrapped in an instance of pni::core::value.

The class currently supports the following types

  • 64Bit integer
  • 64Bit floating point
  • 64Bit complex floats
  • and strings
