nexus::xml::create_from_file()¶

PNINEXUS_EXPORT void pni::nexus::xml::create_from_file(const hdf5::node::Node &parent, const boost::filesystem::path &xml_file)¶

create NeXus objects from an XML file

Reads the content of an XML file and creates NeXus objects below a parent object.

Parameters

  • parent: reference to the parent object

  • xml_file: reference to the path of the XML file
