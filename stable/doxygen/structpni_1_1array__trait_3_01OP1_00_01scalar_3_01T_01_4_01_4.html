Detailed Description

template<typename OP1, typename T>
struct pni::array_trait< OP1, scalar< T > >

This trait determines the common index map and inplace arithmetic implementation types to use for an expression template. This specialization assumes that the LHS is an array while the RHS of the expression is a scalar.
