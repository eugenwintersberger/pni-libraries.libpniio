Detailed Description

template<typename ArrayT, bool is_view>
struct pni::array_view_trait< ArrayT, is_view >

Depending on the index types the ()-operator of mdarray has to either return an instance of array_view or a single value or a reference to a particular element of the array. This template provides the return types for view indexes or and single element indexes. There are two specializations of this template available: one for views and one for single elements.
