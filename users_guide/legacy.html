Legacy formatsÂ¶

The term legacy data refers to all non-Nexus file formats. libpninexus distinguishes between tow families of legacy formats

  • ASCII file where the content is entirely stored in human readable ASCII characters

  • and binary data where the raw binary information is stored in a file.

For details about these different file format types see the following sections.

  • Reading ASCII data
  • Reading binary image data
