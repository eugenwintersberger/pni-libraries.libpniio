Detailed Description

template<typename ITERABLE>
class pni::scalar_iterator< ITERABLE >

<>

This iterator is specifically designed to loop infinitely over a scalar value. As there is no end of sequence and thus no 'end' iterator one cannot use this to loop over a range like this

auto start = infinite_iterator<...>(...);
auto end = infinite_iterator<...>(...);
for(;start != end; ++start)
{
.....
}

! Although the iterator implements the interface of a bidirectional iterator (including all operators) it always refers to the same scalar value.

Template Parameters
ITERABLEthe container to iterate over

◆ operator bool()

template<typename ITERABLE >
pni::scalar_iterator< ITERABLE >::operator bool ( ) const
inlineexplicit

This operator allows the conversion of an iterator to bool. It will return true if the iterator is valid and false otherwise. The iterator is consideres as invalid if its internal state is at least one after the last element or smaller than 0. It is important that this conversion operator is set explicit. Otherwise the iterator would be implicitly convertible to integer (via bool).
