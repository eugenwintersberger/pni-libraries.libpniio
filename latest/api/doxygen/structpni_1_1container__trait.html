Detailed Description

template<typename ContainerT>
struct pni::container_trait< ContainerT >

This type trait collects some information about container types. It can be used for instance in the SFINAE pattern for overloading functions and methods. The information collected includes

  • is_random_access - if true the container provides [] and at() to access data elements
  • is_iterable - if true the container provides an iterator
  • is_contiguous - the data is stored contiguous in memory (for such containers it makes sense to provide a data() method to access this memory
  • is_multidim - the contianer is multidimensional (basically this is only the case for mdarray and array_view.

If a user provides a new container type an overload for this trait be provided along with the implementation.
