template<typename ObjectT >
virtual value_ref pni::array_holder< ObjectT >::operator[] ( size_t  i)
inlinevirtual

Returns a reference to the element at linear index i. The reference is wrapped in the value_ref type erasure. No index checking is performed. If i exceeds the total size of the array a segmentation fault will most probably occur.

Parameters
ilinear index
Returns
reference type erasure.

virtual value pni::array_holder< ObjectT >::operator[] ( size_t  i) const
inlinevirtual

Get value at linear index i. As this is a const member the return value uses the value type erasure to wrap the value returned. This operator does not check the index. If the index is larger than the total size of the array a segmentation fault will most probably occur.
