◆ value()

template<typename ElementT >
std::vector< ElementT > pni::tiff::ifd_entry::value ( std::ifstream &  stream)

Template returns the value of the entry and returs it as a vector of type ElementT. Entries are in general considered as arrays (vectors) in TIFF. Thus the std::vector type is used to represent each entry.

The method makes no assumption about the position of the stream pointer neither does it alter its state.

streaminput stream from which data will be read
