Detailed Description

template<typename MapImpT, size_t... TDimsN>
class pni::static_index_map< MapImpT, TDimsN >

This is the basic static index map. In comparision to index_map its number of elements along each dimensions as well as the number of dimensions cannot be changed after the type has been defined. As all data is static this type is rather efficient as everything resides on the stack. Thus it is perfectly suited for small objects like vectors and matrices.

To create an index map for a 3x3 matrix one could use something like this

typedef static_index_map<c_index_map_imp,3,3> c_matrix_map;

! This class has no special constructors as it is ready to use just be using the default constructor. Unlike index_map this template provides only const iterators (as its dimensions cannot be changed once declared).
