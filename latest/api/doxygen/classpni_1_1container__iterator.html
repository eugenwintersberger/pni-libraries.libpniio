Detailed Description

template<typename IterableT>
class pni::container_iterator< IterableT >

<>

This is the most generic iterator provided by libpninexus. It can be used with all container types provided by the library. A container that wants to use this iterator must implement the following interface

template<typename GeneralT> class Iterable<GeneralT>
{
public:
typedef GeneralT value_type;
size_t size() const;
GeneralT &operator[](size_t i);
GeneralT operator[](size_t i) const;
};

◆ operator bool()

template<typename IterableT >
pni::container_iterator< IterableT >::operator bool ( ) const
inlineexplicit

This operator allows the conversion of an iterator to bool. It will return true if the iterator is valid and false otherwise. The iterator is consideres as invalid if its internal state is at least one after the last element or smaller than 0. It is important that this conversion operator is set explicit. Otherwise the iterator would be implicitly convertible to integer (via bool).
