◆ as()

template<typename ReturnT >
ReturnT pni::value_ref::as

Returns the value of the variable the value_ref instance points to. If the type requested by the user via the template parameter ReturnT is different from the type of the variable value_ref points to the data will be converted if possible. If the conversion fails or is not possible an exception will be thrown.

Exceptions
memory_not_allocated_errorif value is uninitialized
type_errorif data cannot be converted to ReturnT
range_errorif data does not fit into the range of ReturnT
Returns
value of type ReturnT
