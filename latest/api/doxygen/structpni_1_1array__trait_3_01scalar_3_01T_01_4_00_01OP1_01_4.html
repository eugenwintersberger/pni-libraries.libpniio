Detailed Description

template<typename T, typename OP1>
struct pni::array_trait< scalar< T >, OP1 >

This trait determines the common index map and inplace arithmetic implementation types to use for an expression template. This specialization assumes that the LHS is a scalar while the RHS of the expression is an array.
